# Seaborn

## Содержание и алгоритмы (важные файлы)

1) Генерация подземелий (BSP)
   - Файлы: generation\bsp_tree.py, generation\room_generator.py, generation\corridor_generator.py, generation\dungeon.py
   - Описание: рекурсивное BSP-разбиение области на комнаты, затем соединение через коридоры. Разделение по соотношению сторон и минимальной площади.
   - Сложность: средняя O(N log N) по количеству комнат N; построение коридоров O(N).

2) Система лута — Shuffle Bag и Loot Bag
   - Файлы: generation\loot_bag.py, loot\loot_bag.py
   - Описание: "мешок" с весами (shuffle bag) обеспечивает равномерное распределение выпадений при сохранении заданных весов/редкостей.
   - Сложность: заполнение O(B), извлечение O(1).

3) Взвешенный выбор + двоичный поиск
   - Файл: loot\weighted_binary.py
   - Описание: хранит кумулятивные веса и выбирает элемент по случайному числу с помощью двоичного поиска (bisect).
   - Сложность: предрасчёт O(M), выбор O(log M), где M — число вариантов.

4) QuadTree (пространственная индексация)
   - Файл: spatial\quadtree.py
   - Описание: делит пространство на квадранты для ускорения поиска пересечений/коллизий и запросов областей.
   - Сложность: средняя вставка/поиск O(log K), худшая O(K) (при вырожденном распределении), где K — число элементов в узле.

5) Цепи Маркова (генерация текста/сюжета)
   - Файл: ai\markov_chain.py
   - Описание: модель переходов состояний, хранит словарь переходов и кумулятивные вероятности для быстрого сэмплинга.
   - Сложность: построение O(V⋅E), выбор следующего состояния O(log E) (bisect), генерация длины L — O(L log E).

6) Behavior Tree (поведение ИИ)
   - Файл: ai\behavior_tree.py
   - Описание: древовидная структура поведения агентов (последовательности, селекторы, условия). Позволяет декларативно строить сложные ИИ.
   - Сложность: зависит от глубины дерева и числа ветвей; обычно O(nodes) при обходе.

7) Lightmap + Raycasting (динамическое освещение)
   - Файл: lighting\lightmap_raycasting.py
   - Описание: расчёт освещения через отсечение лучей и накопление вкладов в lightmap для источников света.
   - Сложность: приблизительно O(R⋅S) на кадр, где R — число лучей/источников, S — среднее число пересечений.

8) Core — интеграция и World
   - Файлы: core\world.py, core\game.py
   - Описание: фасадные классы World и Game управляют состоянием, порядком обновлений и интеграцией генерации/рендеринга/физики.

## Тесты
- Папка: tests/ и test_bsp.py — содержит юнит-тесты генерации BSP.

## Как запустить
1. Установить зависимости: pip install -r requirements.txt
2. Запустить игру: python main.py
